A work of interactive fiction.

The Cult of Prophets was created with Inform and has IFID 50DA8440-D69E-48AA-844C-81E376193846. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Parchment interpreter. You'll need to have Javascript enabled on your web browser.

Controls (typed commands):

Go [north, south, east, west, up, down]

Take [name of thing]

Examine [name of thing]

Look

Inventory

Save

Drop [item in your inventory]

Talk to [person]

*There are a few more commands required to beat the game that you'll have to figure out for yourself. Also note that not everything has a purpose. Good luck!
